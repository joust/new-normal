Spielanleitung Corona-Uno

Das New-Normal-Uno ist eine Variante des rundenbasierten Uno-Kartenspiels, bei der eine Corona-Diskussion simuliert wird. Es können 2-8 Personen daran teilnehmen, wobei beide Seiten - Idioten und Schafe - gleichermaßen repräsentiert sind.

Es gibt das lokale Spiel - mit 1-7 Computer-Gegnern - und das Netzwerk-Spiel. Das Netzwerk-Spiel läuft über direkte P2P-Kommunikation, kommt also ohne laufenden Kommunikationsserver aus (nur Signaling). Dabei ist ein Spieler der Gastgeber (Host) und die anderen Gäste, die sich mit seiner Spielinstanz verbinden. Das Verbindungsmerkmal ist die Match-ID, die der Gastgeber seinen Gästen mitteilen muss (auf einem anderen Kanal).

Spiel

Die Teilnehmer des Diskurses argumentieren hintereinander durch das Ausspielen von Karten. Ein Zug endet mit dem Ausspielen einer Argument- oder einer Appellier-Karte. Der nächste Spieler an der Reihe muss auf diese letzte gespielte Karte reagieren. Hat er keine passende Antwort, kann er das mit Hilfs- oder Joker-Karten ausgleichen, fehlen ihm auch diese, muss er vom Stapel ein oder mehrere Karten nachziehen bis er eine passende hat.

Kartentypen

Argument-Karte

Eine Argument-Karte repräsentiert ein Thema aus dem Diskurs und alle zugehörigen Argumente einer Seite. Ein initiales Argument wird angezeigt, kann aber vor dem Ausspielen der Karte gewechselt werden. Die Argument-Karte schließt den eigenen Zug ab. Sie kann auch als Antwort auf eine Appellier-Karte gespielt werden. Um auf eine Argument-Karte zu antworten, benötigt man eine Karte mit dem gleichen Thema - oder eine Hilfs- bzw. Joker-Karte. Argument-Karten haben in der Regel eine aufklappbare Rückseite, auf der Quellen aufgeführt sind, die das Argument stützen. Diese können bei Interesse nachgelesen werden.

Appellier-Karte

Die Appellier-Karte repräsentiert eine Referenz auf eine im Diskurs relevante Person oder Institution - mit Authorität oder Bekanntheit - die für die jeweilige Seite steht. Eine initiale Referenz wird angezeigt, kann aber vor dem Ausspielen der Karte gewechselt werden. Die Appellier-Karte schließt den eigenen Zug ab. Sie kann auch als Antwort auf eine Argument-Karte gespielt werden. Die Gegenseite kann auf sie mit einer beliebigen Argument-Karte reagieren, also ggf. das Thema wechseln, oder ebenfalls mit einer Appellier-Karte reagieren. Auch Appellier-Karten haben in der Regel eine aufklappbare Rückseite, auf der Quellen aufgeführt sind. Diese können bei Interesse nachgelesen werden.

Cancel-Karte

Die Cancel-Karte kann als Antwort auf eine Appellier-Karte gespielt werden. Eine initiale Cancel-"Forderung" wird angezeigt, kann aber vor dem Ausspielen der Karte gewechselt werden. Die Cancel-Karte bewirkt, dass die oberste Karte vom Stapel zurück in die Hand des Spieler gelangt, der sie ausgespielt hat, zusammen mit einer Cancel-Karte, so dass dieser Spieler in die Lage versetzt wird, "zurückzuschlagen".

Recherchier-Karte

Die Recherchier-Karte ist eine Joker-Karte und kann auf eine Argument- oder Appellier-Karte gespielt werden. Sie liefert 1-3 passende Karten aus dem eigenen Stapel. Sollte der Stapel keine passenden Karten mehr enthalten, kann sie in seltenen Fällen auch gar keine Karten liefern. Liegt eine Argument-Karte, werden bevorzugt passende Gegen-Argumente geliefert. Finden sich keine oder nicht genug, können auch Appellier-Karten dazukommen. Liegt oben auf dem Stapel eine Appellier-Karte, so werden Appellier-Karten geliefert. Finden sich keine oder nicht genug, können auch Argument-Karten dazukommen. Die Karten werden der eigenen Hand hinzugefügt und können im Anschluss ausgespielt werden.

Strohmann-Karte

Die Strohmann-Karte ist eine Joker-Karte und kann auf eine Argument-Karte gespielt werden. Sie hebt die vom Gegner gespielte Karte auf und ermöglicht es, sie durch eine andere zu ersetzen (dem Gegner als etwas in den Mund zu legen). Dies geschieht, indem beim Ausspielen der eigenen Hand eine passende Karte aus dem gegnerischen Stapel hinzugefügt wird. Diese muss der Spieler dann Ausspielen und mit einer eigenen Argument- oder Appellier-Karte beantworten, um den Zug zu beenden.

Allgemeinplatz-Karte

Die Allgemeinplatz-Karte ist eine Hilfs-Karte, mit der der Diskurs zu den eigenen Gunsten beeinflusst werden kann. Eine initialer Allgemeinplatz wird angezeigt, kann aber vor dem Ausspielen der Karte gewechselt werden. Mit der Allgemeinplatz-Karte kann auf eine Argument- oder Appellier-Karte geantwortet werden. Sie hebt den Zwang auf, genau auf diese Karte und ihr Thema reagieren zu müssen. Mit ihr wird ein Themenwechsel ermöglicht. Von Allgemeinplatz-Karten können mehrere hintereinander ausgespielt werden, der Zug ist aber erst beendet, wenn der aktuelle Spieler abschließend eine Argument-, Appellier- oder Diskutier-Karte ausgespielt hat.

Etikettier-Karte

Die Ettiketier-Karte ist eine Hilfs-Karte, mit der der Diskurs zu den eigenen Gunsten beeinflusst werden kann. Eine initiales Ettikett wird angezeigt, kann aber vor dem Ausspielen der Karte gewechselt werden. Mit der Etikettier-Karte kann auf eine Argument- oder Appellier-Karte geantwortet werden. Sie hebt den Zwang auf, genau auf diese Karte und ihr Thema reagieren zu müssen. Mit ihr wird ein Themenwechsel ermöglicht. Von Etikettier-Karten können mehrere hintereinander ausgespielt werden, der Zug ist aber erst beendet, wenn der aktuelle Spieler abschließend eine Argument-, Appellier- oder Diskutier-Karte ausgespielt hat.

Ausdiskutier-Karte

Die Ausdiskutier-Karte ist eine Hilfs-Karte, mit der der Diskurs auf ein Thema gelenkt werden kann. Eine initiales Thema wird angezeigt, kann aber vor dem Ausspielen der Karte gewechselt werden. Die Ausdiskutier-Karte beendet den eigenen Zug. Mit ihr kann man dafür sorgen, dass das gespielte Thema von allen Beteiligten ausdiskutiert werden muss. Sie kann z.B. eingesetzt werden, wenn einzelne Spieler kurz vor dem Gewinn (durch Ausspielen all ihrer Karten) stehen. Durch ihr Ausspielen erhalten alle Spieler eine spezielle "unendliche" Argument-Karte, die so lange in der Hand der Beteiligten bleibt, bis das Thema zu Ende diskutiert wurde - oder das Ausdiskutier-Thema durch Ausspielen einer neuen Ausdiskutier-Karte auf ein anderes Thema gelenkt wurde.

Verbannen-Karte

Mit der Verbannen-Karte kann man dafür sorgen, dass das Thema der zuletzt ausgespielten Karte aus dem Spiel verbannt wird. Alle Argument-Karten zum Thema verschwinden nach ihrem Ausspielen aus den Stapeln und den Händen der Spieler. Der aktuelle Spieler kann mit einer beliebigen eigenen Karte fortsetzen. Die Verbannen-Karte kann z.B. direkt oder verzögert gegen eine Ausdiskutier-Karte eingesetzt werden. Strategisch sinnvoll kann es sein, ein Thema zu verbannen, zu dem der Argumente-Pool der eigenen Seite besonders groß ist.

Aussetzen-Karte

Mit der Aussetzen-Karte kann sich der aktuelle Spieler die Möglichkeit verschaffen, auszusetzen, d.h. keine Argument- oder Appellier-Karte auf dem Stapel ablegen zu müssen. Statt dessen ist sofort der nächste Spieler der gleichen Seite an der Reihe. Dies kann sinnvoll sein, wenn man langes Nachziehen vermeiden will. Die Aussetzen-Karte ist im Spiel ab 4 Teilnehmern. In einer solchen Konstellation diskutieren beide Seiten nicht mehr allein und es kann an einen Mitspieler der gleichen Seite weitergegeben werden.

Spielende

Das Spiel endet in dem Moment, in dem eine Seite keine Karten mehr auf der Hand hat - sie ist der Gewinner der Diskussion, weil sie alle Argumente losgeworden ist. Bei mehr als zwei Spielern kann dies dazu führen, dass in einem Team weniger Leute spielen als im anderen!
