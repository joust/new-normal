Installation

New Normal ist eine so genannte PWA, eine Progressive Web App. Das bedeutet, Du kannst das Spiel als App auf Deinem Handy oder Tablet installieren - ganz ohne App Store- mit Hilfe der folgenden Anleitung für Dein Mobil-Betriebssystem.

iOS

  1. Starte Safari auf Deinem iPhone oder iPad.
  2. Gib die Adresse new-normal.app ein.
  3. Tippe unten mittig auf das Freigabesymbol (Quadrat mit dem Pfeil nach außen).
  4. Scrolle zur Liste der Aktionen und tippe auf Zum Startbildschirm hinzufügen. Safari fügt New Normal dann zu Deinem Startbildschirm hinzu.

Android

  1. Starte die Browser-App „Chrome“.
  2. Gib die Adresse new-normal.app ein.
  3. Tippe auf das Menüsymbol (3 Punkte oben rechts) und dann auf Zum Startbildschirm hinzufügen.
  4. Gib Deinen Wunschnamen für die Verknüpfung ein. Chrome fügt New Normal dann zu Deinem Startbildschirm hinzu.
