Spielanleitung

Das New-Normal-Uno ist eine Variante des rundenbasierten Uno-Kartenspiels, bei der eine Corona-Diskussion simuliert wird. Es können 2-10 Personen daran teilnehmen, wobei beide Seiten - Idioten und Schafe - gleichermaßen repräsentiert sind.

Es gibt das lokale Spiel - gegen den Computer - und das Netzwerk-Spiel. Das Netzwerk-Spiel läuft über direkte P2P-Kommunikation, kommt also ohne laufenden Kommunikationsserver aus (nur Signaling). Dabei ist ein Spieler der Gastgeber (Host) und die anderen Gäste, die sich mit seiner Spielinstanz verbinden. Das Verbindungsmerkmal ist die Match-ID, die der Gastgeber seinen Gästen mitteilen muss (auf einem andern Kanal).

Spiel

Die Teilnehmer des Diskurses argumentieren hintereinander durch das Ausspielen von Karten. Ein Zug endet immer automatisch mit dem Ausspielen einer Argument- oder einer Appellier-Karte. Der nächste Spieler an der Reihe muss dann auf diese letzte gespielte Karte reagieren. Hat er keine passende Antwort, kann er das mit Spezial-Karten ausgleichen, fehlen ihm auch diese, muss er vom Stapel ein oder mehrere Karten nachziehen bis er eine passende hat.

Kartentypen

Argument-Karte

Eine Argument-Karte repräsentiert eines von vielen Argumenten aus dem Diskurs. Sie gehört immer zu einem Thema und um auf sie zu antworten, benötigt man eine Karte mit dem gleichen Thema. Argument-Karten haben idR eine aufklappbare Rückseite, auf der Quellen aufgeführt sind, die das Argument stützen. Diese können bei Interesse nachgelesen werden. Alternativ zu einer Argumentkarte kann immer eine Appellier-Karte gespielt werden.

Appellier-Karte

Die Appellier-Karte repräsentiert eine Authorität oder Berühmtheit, die für die jeweilige Seite steht. Das können Personen und Institutionen sein. Sie kann ein Argument ersetzen. Die Gegenseite kann auf sie mit einer beliebigen Argument-Karte reagieren, also ggf. das Thema wechseln, oder ebenfalls mit einer Appellier-Karte reagieren. Auch Appellier-Karten haben idR eine aufklappbare Rückseite, auf der Quellen aufgeführt sind. Diese können bei Interesse nachgelesen werden.

Recherchier-Karte

Die Recherchier-Karte ist eine Joker-Karte und kann auf eine Argument- oder Appellier-Karte gespielt werden. Sie liefert 1-3 passende Karten aus dem eigenen Stapel. Sollte der Stapel keine passenden Karten mehr enthalten, kann sie in seltenen Fällen auch gar keine Karten liefern. Liegt eine Argument-Karte, werden bevorzugt passende Gegen-Argumente geliefert. Finden sich keine oder nicht genug, können auch Appellier-Karten dazukommen. Liegt oben auf dem Stapel eine Appellier-Karte, so werden Appellier-Karten geliefert. Finden sich keine oder nicht genug, können auch Argument-Karten dazukommen. Die Karten werden der eigenen Hand hinzugefügt und können im Anschluss ausgespielt werden.

Strohmann-Karte

Die Strohmann-Karte ist eine Joker-Karte und kann auf eine Argument-Karte gespielt werden. Sie hebt die vom Gegner gespielte Karte auf und ermöglicht es, sie durch eine andere zu ersetzen (dem Gegner als etwas in den Mund zu legen). Dies geschieht, indem beim Ausspielen der eigenen Hand eine passende Karte aus dem gegnerischen Stapel hinzugefügt wird. Diese muss der Spieler dann Ausspielen und mit einer eigenen Argument- oder Appellier-Karte beantworten, um den Zug zu beenden.

Allgemeinplatz-Karte

Mit der Allgemeinplatz-Karte kann auf eine Argument- oder Appellier-Karte geantwortet werden. Sie hebt den Zwang auf, genau auf diese Karte und ihr Thema reagieren zu müssen. Mit ihr wird ein Themenwechsel ermöglicht. Von Allgemeinplatz-Karten können mehrere hintereinander ausgespielt werden, der Zug ist aber erst beendet, wenn, denn der aktuelle Spieler abschließend eine Argument- oder Appellier-Karte ausgespielt hat.

Etikettier-Karte

Mit der Etikettier-Karte kann auf eine Argument- oder Appellier-Karte geantwortet werden. Sie hebt den Zwang auf, genau auf diese Karte und ihr Thema reagieren zu müssen. Mit ihr wird ein Themenwechsel ermöglicht. Von Etikettier-Karten können mehrere hintereinander ausgespielt werden, der Zug ist aber erst beendet, wenn, denn der aktuelle Spieler abschließend eine Argument- oder Appellier-Karte ausgespielt hat.

Ausdiskutier-Karte

Mit der Ausdiskutier-Karte kann man dafür sorgen, dass das Thema der Karte von allen Beteiligten ausdiskutiert wird. Sie kann z.B. eingesetzt werden, wenn einzelne Spieler kurz vor dem Gewinn (durch Ausspielen all ihrer Karten) stehen. Durch ihr Ausspielen erhalten alle Spieler eine spezielle "unendliche" Argument-Karte, die so lange in der Hand der Beteiligten bleibt, bis das Thema zu Ende diskutiert wurde.

Aussetzen-Karte

Die Aussetzen-Karte ist im Spiel ab 4 Teilnehmern. In einer solchen Konstellation diskutieren beide Seiten nicht mehr allein. Mit dieser Karte besteht die Möglichkeit, auszusetzen, d.h. keine Argument- oder Appellier-Karte auf dem Stapel ablegen zu müssen. Statt dessen ist sofort der nächste Spieler der gleichen Seite an der Reihe. Dies kann sinnvoll sein, wenn man langes Nachziehen vermeiden will.

Spielende

Das Spiel endet in dem Moment, wo ein Spieler keine Karten mehr auf der Hand hat. Er ist dann der Sieger der Diskussion, da er alle seine Argumente losgeworden ist.

Ideen

Bei mehr als 2 Mitspielern könnte es (zusätzliche) Karten geben, die die Zusammenarbeit unterstützen oder zum "Angriff" auf den Gegner abseits der Argumente (unter der Gürtellinie?) genutzt werden können. Oder bestehende Karten könnten anders funktionieren.
